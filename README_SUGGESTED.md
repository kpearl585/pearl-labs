# Pearl Labs

**A modern consultancy for defense technology and data-driven decision making.**

Pearl Labs specializes in:
- Combat forecasting and wargaming systems
- Geospatial intelligence platforms
- Data fusion and analysis tools
- Military decision support software

## Projects

- **ARGUS** — Tactical forecasting platform using TNDM v6.4 combat modeling
- **FenceOS** — Construction estimation and margin protection software
- Custom defense technology consulting

## This Website

Built with Next.js 14, this site showcases Pearl Labs' services, advisory capabilities, and project portfolio.

### Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Structure

```
app/
├── components/     # Reusable UI components (Hero, Services, etc.)
├── blog/          # Blog content
├── estimate/      # FenceOS estimate calculator
├── work/          # Portfolio case studies
└── api/           # API routes
```

## Development

This project uses:
- [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) with Geist font
- Server-side rendering for SEO
- Responsive design (mobile-first)

## Deployment

Deployed automatically via Vercel on push to `main`.
